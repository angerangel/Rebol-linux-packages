Rebol-linux-packages

All files and script nedded to create linux installation packages for Rebol

You can download packages from: http://www.maxvessi.net/rebsite/Linux/index.php https://github.com/angerangel/Rebol-linux-packages

To update the packages, update the rebol interpeter in the folders: rebol/usr/bin/ rebol_AMD64/usr/bin/

then launch packcreator.sh script.
